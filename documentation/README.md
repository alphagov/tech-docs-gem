# Tech Docs Template Documentation

This directory contains the documentation for the [`tech-docs-template`](https://github.com/alphagov/tech-docs-template), which uses the gem in this repository.

The documentation covers:

- creating and updating a site with the tech docs template
- examples of how to use the gem helpers

To start the documentation in this repo, run:

```sh
cd documentation
bundle install
bundle exec middleman server
```

## Publishing changes

GitHub Actions automatically publishes changes merged into the main branch of this repository to [https://alphagov.github.io/tech-docs-gem/](https://alphagov.github.io/tech-docs-gem/).
